Веб-страница для ведения трекера с книгами и статьями

Реализованные функции:
1. Добавление, редактирование и удаление книг и статей.
2. Фильтрация по статусу прочтения и теме/жанру.
3. В списках книг и статей сначала идут объекты со статусом "В процессе", после объекты со статусом "Прочитана".
4. Отдельно идёт список наверху страницы с избранными книгами и статьями. Свойство избранности можно менять.
5. Реализована пагинация и добавлена админка, через которую добавлены жанры и темы для книг и статей.

Есть страница с возможностью входа в систему, но сейчас сайт работатет только через админку, поэтмоу лучше на странице /admin залогиниться и проверять уже оттуда работоспособность сайта.
